How It Works

To use RoutifyMe, log in or create a new account, navigate to the "Route Optimization" section, enter your start location and end location, along with any intermediate stops, click the "Optimize Route" button, and view your optimized route on the interactive map.

Frequently Asked Questions

How does RoutifyMe optimize routes?

RoutifyMe uses advanced algorithms and Bing Maps data to calculate the most efficient route based on your start and end locations and any intermediate stops.

Can I add new locations directly on the map?

Yes, you can add new locations by clicking directly on the interactive map in the RoutifyMe app.

How do I get started with RoutifyMe?

To get started, clone the RoutifyMe repository, install the dependencies, set up and activate the virtual environment, create a .env file, run the application, and visit http://localhost:8000 in your web browser.

Can I use my current location as a starting point?

Yes, RoutifyMe supports user location search, allowing you to use your current location as a starting point for route optimization.
